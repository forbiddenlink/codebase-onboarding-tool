# CodeCompass VS Code Extension

AI-powered codebase onboarding directly in VS Code.

## Features

- **Repository Analysis**: Analyze your codebase to understand its structure
- **AI Chat**: Ask questions about your code in natural language
- **Inline Annotations**: See helpful tips and explanations as you code
- **Hover Tooltips**: Get AI-powered insights when hovering over code

## Getting Started

1. Install the extension
2. Open a repository in VS Code
3. Run the command "CodeCompass: Analyze Repository"
4. Start exploring!

## Configuration

- `codecompass.apiKey`: Your Anthropic API key (required for AI features)
- `codecompass.enableHoverTooltips`: Show tooltips on hover (default: true)
- `codecompass.enableInlineAnnotations`: Display inline annotations (default: true)

## Requirements

- VS Code 1.85.0 or higher
- Anthropic API key for AI features
