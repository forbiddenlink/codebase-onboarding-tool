# Changelog

All notable changes to CodeCompass will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Core repository analysis engine with tree-sitter parsing
- AI-powered chat interface with Claude Sonnet 4
- Modern web dashboard with Next.js 14 and Tailwind CSS
- Database layer with Prisma ORM and SQLite
- CLI tool structure for command-line operations
- VS Code extension scaffolding
- Comprehensive test suite (200+ test cases defined)
- Command palette for quick actions
- Notification system
- Custom design system with warm coral/amber/teal palette

### Changed
- Updated TypeScript to strict mode across all packages
- Improved error handling in file parsers
- Enhanced type safety throughout codebase

### Fixed
- TypeScript compilation errors in web package
- Missing metadata field in FileNode schema
- Unused imports and variables warnings
- Framer Motion type conflicts in GlassCard component

### Removed
- Temporary session notes and code review documents
- Development test files from root directory
- Obsolete build reports and progress tracking files

## [0.1.0] - 2025-02-16

### Added
- Initial project structure and monorepo setup
- Feature specification document (200+ test cases)
- Development environment configuration
- README with comprehensive documentation
- Project roadmap and contribution guidelines
